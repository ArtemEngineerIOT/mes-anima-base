# UC-08 — Этикетка рулона

## Связь с другими требованиями

Выглядит как часть **US-13** / **US-15**. Вероятнее всего будет детализирована там; окончательное решение — после обсуждения этих UC.

На текущий момент сценарий рассматривается как часть блока **«Выпуск»**: оператор завершает намотку рулона, печатает этикетку из интерфейса выпуска и затем регистрирует выпуск.

## Назначение

Обеспечить печать этикетки рулона.

## Акторы

- **Основной актор**: оператор (любого участка)
- **Второстепенные акторы**: принтер этикеток, начальник смены

## Предусловия

1. Процесс производства рулона завершён; производится выпуск полуфабрикатной продукции.
2. В MES доступны накопленные данные по рулону, необходимые для печатной формы: заказ, метраж, вес нетто, вес брака при наличии и данные для штрихкодов.

## Основной поток событий (печать единой этикетки)

1. Оператор (или помощник оператора) завершает намотку рулона и переходит к интерфейсу печати этикетки в MES (блок **«Выпуск»**).
2. Система автоматически формирует печатную форму этикетки, включающую накопленные данные по рулону: заказ, метраж, вес нетто, вес брака (если был зафиксирован); формируется набор штрихкодов.
3. Оператор нажимает кнопку **«Печать этикетки»**.
4. MES выводит на экран PDF-документ этикетки с текстовыми данными и штрихкодами.
5. Оператор клеит этикетку на рулон.
6. Оператор регистрирует выпуск.
7. Система блокирует возможность редактирования полей данной этикетки (вес, сорт, дата) для учётной записи оператора.

## Альтернативные потоки

### A1 — Исправление после покидания локации

Исправление ошибки при печати этикетки выполняется через начальника смены или администратора.

## Постусловия

1. На рулон наклеена этикетка.
2. Данные в MES и 1С синхронизированы.
3. Изменение данных постфактум невозможно без дополнительного уровня авторизации.
