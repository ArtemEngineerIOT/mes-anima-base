# UC-13 — Материалы. Списание/возврат

## Назначение

Фиксация расхода сырья и полуфабрикатов в производстве с возможностью:

- полного/частичного списания;
- указания категории расхода;
- обеспечения прослеживаемости.

## Акторы

- **Основной актор**: Оператор ОГП
- **Второстепенные акторы**: MES (интерфейс), 1С (учёт), Начальник смены (консультации при нештатных ситуациях)

## Предусловия

1. Оператор аутентифицирован.
2. Заказ на участке активен.
3. На складе у машины находятся рулоны материалов, предназначенные для данного заказа.
4. В DWH доступен справочник серий материалов с закупочным коэффициентом (грамматурой) для точного пересчёта метров в килограммы.

## Основной поток событий

1. Оператор открывает экран выполнения этапа, блок **«Материалы. Списание/возврат»**.
2. Сканирует штрих-код рулона (или вводит номер вручную).
3. MES выполняет проверки:
   - рулон не заблокирован;
   - материал разрешён для данного заказа (основной или альтернативный);
   - локация рулона входит в список разрешённых для списания (на основе справочника подразделений из DWH).
4. Система отображает информацию о серии: **номенклатура, серия, единица измерения, метраж, вес**.
5. Оператор выбирает тип операции:
   - **Отразить возврат**: выполняет два действия в 1С — частичное списание рулона (на указанное израсходованное количество) и оформление возврата остатка на выбранный склад.
   - **Списать полностью**.
   - **Завершить списания по этапу**: кнопка сигнализирует 1С, что по этому этапу больше не будет списаний, и 1С выполняет нормативное списание ЛКМ пропорционально плёнке.
   - **Печать этикетки**.
6. При нажатии кнопок:
   - **«Отразить возврат»** (оператор указывает потраченное количество — метры или килограммы — и склад возврата из выпадающего списка),
   - **«Списать полностью»**,
   - **«Завершить списание»**  
   MES формирует JSON-запрос к 1С, содержащий:
   - номер этапа;
   - штрих-код (серию);
   - тип операции (`full` / `partial`);
   - для полного списания — признак «списать полностью»;
   - для частичного — возвращаемое количество и склад возврата.
7. Возврат всей серии целиком разрешён только выполнением операции **«Перемещение»** (UC-22) и в этом процессе не предусмотрен.
8. 1С выполняет операцию:
   - при полном списании — рулон списывается целиком (независимо от показаний датчика);
   - при частичном — создаются две операции: списание потреблённой части и возврат остатка на указанный склад.
9. 1С возвращает результат. MES отображает подтверждение и обновляет таблицу выполненных операций.
10. При нажатии кнопки **«Печать этикетки»** открывается печатная форма возвратной этикетки. Форма возвратной этикетки зависит от списываемой номенклатуры и определяется автоматически в момент списания.

## Альтернативные потоки

### A1 — Частичное списание с автоматическим пересчётом

1. Оператор вводит количество в метрах — система автоматически рассчитывает вес по закупочному коэффициенту серии (грамматура).
2. Закупочный коэффициент берётся из справочника серий в DWH.

### A2 — Ошибка обработки в 1С

См. UC-23.

## Постусловия

1. В 1С зарегистрирован факт списания/возврата материала (полного или частичного).
2. Остатки материалов обновлены.
3. История операций доступна для просмотра.
