# Материалы. Перемещение между складами

| Поле | Значение |
|------|----------|
| **Назначение** | Предоставить оператору возможность перемещать материалы (рулоны / серии) между складами, производственными машинами и другими локациями без привязки к конкретному производственному этапу |
| **Основной актор** | Оператор |
| **Второстепенные акторы** | MES (интерфейс), 1С (учёт), начальник смены (консультации при нештатных ситуациях) |

## Предусловия

1. Пользователь аутентифицирован в MES и имеет право на выполнение операций перемещения.
2. В DWH доступны данные об остатках материалов на всех складах (номенклатура, серия, количество, статус блокировки, срок годности).

## Основной поток событий

1. Оператор открывает экран **«Перемещение материалов»** (доступ из главного меню).
2. Система отображает два выпадающих списка:
   - **Склад-отправитель** — список доступных складов / машин;
   - **Склад-получатель** — список всех складов, куда разрешено перемещение.
3. После выбора склада-отправителя в табличной части автоматически отображаются все рулоны (серии), числящиеся на этом складе.

   Колонки таблицы: **номенклатура**, **назначение**, **серия**, **количество**, чекбокс для выбора.

4. Оператор отмечает чекбоксами один или несколько рулонов, которые требуется переместить.
5. Оператор при необходимости указывает:
   - желаемое время доставки (дата / время);
   - комментарий для склада (свободный текст).
6. Оператор нажимает кнопку **«Переместить»**.
7. MES формирует JSON-запрос к 1С типа «Перемещение», содержащий:
   - склад-отправитель;
   - склад-получатель;
   - список выбранных серий;
   - желаемое время доставки;
   - комментарий.
8. 1С выполняет проверку (достаточность остатков, отсутствие блокировок) и создаёт документ **«Перемещение товаров»**. В ответе возвращает статус операции.
9. Обработка ответа:
   - **Успех:** 1С возвращает подтверждение с номером документа перемещения. MES обновляет таблицу остатков и показывает уведомление «Перемещение оформлено».
   - **Ошибка** (недостаточно остатка, рулон заблокирован, склад не найден и т. п.): 1С возвращает структурированное сообщение об ошибке. MES отображает его оператору, операция не выполняется.

## Альтернативные потоки

### А1: Ошибка связи

См. **US-12.A1**.

## Постусловия

1. В 1С создан документ «Перемещение товаров».
2. Остатки материалов на складе-отправителе уменьшены, на складе-получателе — увеличены.
3. В MES обновлены данные о складских остатках.
